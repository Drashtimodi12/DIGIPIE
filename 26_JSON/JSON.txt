What is JSON(JavaScript Object Notation)? 
    It is a lightweight, text-based format used for storing and transferring data between applications.
    JSON is easy for humans to read and easy for machines to parse.
    A structured text format similar to Python dictionaries

Before using JSON, import the module:       """import json"""

It is mainly used to exchange data between:
    Frontend ↔ Backend
    Server ↔ Client
    Web apps, Mobile apps, APIs, Databases

Why is JSON used?
    Simple and lightweight; Language-independent; Works on all platforms; Most APIs use JSON; 
    Stores configuration data; Easy to convert into Python objects

Basic JSON Rules
    Data is written in key–value pairs; Keys must be inside double quotes ("")
    Example:    {"name": "Drashti", "age": 22}
    Values can be:
        String → "Drashti"
        Number → 22
        Boolean → true, false
        Null → null
        Array → [1, 2, 3]
        Object → {"x": 10, "y": 20}
