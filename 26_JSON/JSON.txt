What is JSON(JavaScript Object Notation)? 
    It is a lightweight, text-based format used for storing and transferring data between applications.
    JSON is easy for humans to read and easy for machines to parse.
    A structured text format similar to Python dictionaries

Before using JSON, import the module:       """import json"""

It is mainly used to exchange data between:
    Frontend ↔ Backend
    Server ↔ Client
    Web apps, Mobile apps, APIs, Databases

Why is JSON used?
    Simple and lightweight; Language-independent; Works on all platforms; Most APIs use JSON; 
    Stores configuration data; Easy to convert into Python objects

Basic JSON Rules
    Data is written in key–value pairs; Keys must be inside double quotes ("")
    Example:    {"name": "Drashti", "age": 22}
    Values can be:
        String → "Drashti"
        Number → 22
        Boolean → true, false
        Null → null
        Array → [1, 2, 3]
        Object → {"x": 10, "y": 20}

Useful json module functions
        Function	        Use
    json.dumps()	Convert Python to JSON string
    json.loads()	Convert JSON string to Python
    json.dump()	    Write JSON directly to a file
    json.load()	    Read JSON directly from a file



Parameters for json.dumps() and json.dump()
(Used when converting Python → JSON)
    Parameter	                    Meaning
    indent	                Pretty formatting (adds spaces/newlines)
    sort_keys	            Sorts dictionary keys alphabetically
    ensure_ascii	        False = allow Unicode (Hindi, Gujarati, emoji)
    skipkeys	            Ignore keys that are not strings
    default	                Convert unsupported objects (custom function)
    separators	            Customize separators like ("," , ":")
    cls	                    Use custom JSON encoder class (JSONEncoder)
    allow_nan	            Allow NaN, Infinity, -Infinity values
    check_circular	        Detect circular references (usually keep True)
    fp (in dump only)	    File pointer to write JSON into


Parameters for json.loads() and json.load()
(Used when converting JSON → Python)
        Parameter	        Meaning
    object_hook	            Modify each dict after decoding
    object_pairs_hook	    Preserve order of pairs (OrderedDict)
    parse_int	            Custom function for integers
    parse_float	            Custom function for floats
    parse_constant	        Handle NaN, Infinity, -Infinity
    strict	                False = allow trailing commas
